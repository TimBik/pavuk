                     <h3>Подсветка синтаксиса</h3>
                     <p>Настраивается через интерфейс программы. Эта функция позволяет выделять разными цветами синтаксические конструкции, тем самым облегчает визуальное восприятие текста. Основные возможности: формирование списка ключевых слов, сворачивание синтаксических блоков, определение формата чисел, настройка стилей. Также подсветка проверяет, закрыты ли теги в тексте.</p>
                     <h3>Сворачивание кода</h3>
                     <p>Программа позволяет скрыть фрагмент кода, оставив только название функции.</p>
                     <h3>Автодополнение</h3>
                     <p>Вводит дополнение теста по введенной части, их описание, подсказки параметров функций. Для верной работы необходимо, чтобы строки были отсортированы в алфавитном порядке в xlm.</p>
                     <h3>Простая смена кодировки</h3>
                     <p>Сначала нужно вырезать выделенный код, затем кликнуть «Меню → Кодировки», выбрать кодировку вставить в код буфера обмена. Кроме того, если при печати текст выведен не в том коде, в настройках печати его можно поменять.</p>
                     <h3>Основные функции</h3>
                     <ul>
                        <li>Встроенный FTP-клиент позволяет без других программ скачивать файлы, редактировать их и закачивать на хостинг;</li>
                        <li>Менеджер проектов;</li>
                        <li>Быстрое перетаскивание фрагментов кода с помощью выделения, перетаскивание ярлыков документов в редактор, чтобы открыть их;</li>
                        <li>Мультистрочное редактирование;</li>
                        <li>Использование горячих клавиш, по двойному клику можно перейти в место ошибки;</li>
                        <li>Резервное копирование файлов;</li>
                        <li>Запись макросов и их выполнение;</li>
                        <li>Карта документа;</li>
                        <li>Блоковое выделение текста, выделение нескольких разных блоков;</li>
                        <li>Масштабирование.</li>
